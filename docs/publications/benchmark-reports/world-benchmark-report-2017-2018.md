# World Benchmark Report 2017 - 2018

## Summary

The UBI Global World Benchmark Report 2017-2018 presents a large-scale study of university-linked business incubators and accelerators, benchmarking 259 programs across 53 countries.

The report examines how these programs create value for ecosystems, client startups, and the incubation programs themselves, while also highlighting best practices, methodology, and participating-program data.
