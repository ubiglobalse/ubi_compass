# World Benchmark Report 2019 - 2020

## Summary

The UBI Global World Benchmark Report 2019-2020 presents a broad assessment of incubators and accelerators worldwide, benchmarking 364 programs across 82 countries.

The report analyzes performance across ecosystem value, client startup success, and program excellence, offering a data-driven reference for innovation leaders, universities, policymakers, and ecosystem builders.
