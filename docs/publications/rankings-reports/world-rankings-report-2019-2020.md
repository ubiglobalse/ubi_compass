# World Rankings Report 2019 - 2020

## Summary

The UBI Global World Rankings Report 2019-2020 highlights high-performing incubators and accelerators from 364 programs across 82 countries, showing how these organizations contribute to entrepreneurship, job creation, and broader economic growth.

This edition expands beyond university-linked programs to include public, private, and corporate incubators and accelerators, giving readers a broader view of global innovation performance and ranking methodology.
