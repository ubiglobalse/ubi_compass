# World Rankings Report 2021 - 2022

## Summary

The UBI Global World Rankings Report 2021-2022 highlights high-performing incubators and accelerators worldwide based on data from 109 programs across 56 countries.

Built on the World Benchmark Study, the report focuses on value creation, startup success, ecosystem impact, and additional recognition for ESG-related excellence.
